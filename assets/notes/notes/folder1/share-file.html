+r+Note: This API is available if *activity* and *files* is set to true in *includeApi*.+r+ *android.activity.shareFile(file,mimeType,callback)* Share a file. Starts an activity for sharing a file. *file* - the file to share. It may be an android.File object, or a string that contains a file path. *mimeType* - a string that contains the mime type for the data. Example: "video/mp4" or "image/png" *callback* - the function to call when the share activity finishes. This is optional. <script> function shareMyFiles(file,mimeType,callback) { android.activity.shareFile(file,mimeType,callback); } </script> To test this, put an image (photo.jpg or any) in the "Download" folder of your phone. <script> function sharePhoto() { var dir=android.files.getPublicDir("downloads"); var file=new android.File(dir,"photo.jpg"); android.activity.shareFile(file,"image/jpg"); } </script> <body> <button onclick="sharePhoto()">Share my photo</button> </body> function shareHtml(html) { var intent={ action:"android.intent.action.SEND", type:"text/html", extras:{ "android.intent.extra.TEXT" : html } }; android.activity.startActivity(intent,true); }
