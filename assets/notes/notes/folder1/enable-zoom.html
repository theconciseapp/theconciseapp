+r+Note: This API is available if *webView* is set to true in *includeApi*.+r+ Use this if you want your user to be able to zoom the content of your page *enableZoom(true)* Enables or disables the zoom feature. If true, the zoom feature will be enabled; otherwise it will be disabled. <script>android.webView.enableZoom(true);</script> You can also set this in the *App tree* by clicking your *WebView | mywebview* node, click on the plus icon and search for *zoom* [img=file:///assets/notes/notes/folder1/photos/zoom.png][/img] To disable zooming any time, you can call: <script> android.webView.enableZoom(false); </script>
