A toast is a view containing a quick little message for the user. It disappears by itself after a few seconds. Note: This API is available if *toast* is set to true in *includeApi*. *show(text)* _Show a toast._ *text* - the text to show. android.toast.show("Login failed. Try Again"); Result: [img=file:///assets/notes/notes/folder1/photos/toast.png][/img] *showLong(text)* _Show a toast for a longer time._ *text* - the text to show. android.toast.showLong("This is a long text that takes more time to read.");
