+r+Note: This API is available if *activity* is set to true in *includeApi*.+r+ Use this function to share or send any type of text to another app on the device or bluetooth <script> function shareText(text, mimeType) { mimeType=mimeType||"text/html"; var intent={ action:"android.intent.action.SEND", type: mimeType, extras:{ "android.intent.extra.TEXT" : text } }; android.activity.startActivity(intent,true); } </script> Test: <body> <button onclick="shareText('I am beautiful')">Share sample text</button> </body>
