In this lesson, i will show you how to change the texts colour of your webview page (hello.html). When you created your app, the text's colour is black. That is the default color until it's changed. You +r+cannot+r+ change the colour of text of a WebView page through the *app tree*. This is why it would be an added advantage if you know or learn HTML and CSS +s+( https://www.w3schools.com )+s+ App Builder guide is not meant to teach you HTML, CSS, JAVASCRIPT or JQUERY, we will only make use of them only where and when necessary. To change the text colour from black to white: 1. Open *Assets* under *App Files* of your app 2. Open *hello.html* In between your <style> </style>, enter h1{ color: white; } or body{ color: white; } Instead of colour name (white;), you can also use Css color code: "*#ffffff;*" which also refers to white +s+( _learn more from_ https://www.w3schools.com )+s+ _Full code +s+(you may click code below to copy it)+s+_ <html> <style> h1{ color: white; } </style> <script src="android.js"></script> <script> </script> <body> <h1>Hello <br>my name is so so so <br>I'm beautiful</h1> </body> </html> 3. Build and launch *Our text colour has been successfully changed to white.* Watch video for clearer implementation [youtube=gW5ANwUA6_o]Change text colour of a WebView page~[/youtube] Explanation: <h1> is an HTML tag that stands for header. There are six types of html headers: h1, h2, h3, h4, h5, h6. And as you can see, we put _Hello, i'm so so so i'm beautiful_ in-between <h1> </h1> <style> </style> is a css tag used inside an html page. All css styles are put inside or in-between <style> </style> Therefore, what this css style "_h1{ color: white; }_" does is that it targets all <h1> html tags in our html page (hello.html) and changed the colour of texts in-between <h1> </h1> to white. Also note that in our style, we used "color" not "colour" i.e color: white;
